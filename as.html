Artisan Style

Artisan Style is conceptual on-line clothing company.

Role

To design a visually appealing and engaging digital experience that creates an easy selection and checkout process for the user, thus compelling repeat business. The project is intended to focus on the e-commerce experience.

Technology

Illustrator / Balsamiq / InVision / Peek User Testing / Sketch

PROJECT APPROACH

Map out the selection process focusing on ease of targeting items to be considered.
Design visual experience....

WIREFRAMES AND CLICK STREAM

Converting sketches to low fidelity wireframes crystalized ... I experimented with creating low fidelity wireframes in Sketch, thinking that the conversion to hi-fidelity would be sped up. It slowed down the initial layout part of the design process and created this hybrid low/hi fidelity output. I created a prototype with InVision utilizing these part low-fidelity wireframe/part hi-fidelity wireframes. When I started getting user experience feedback, via Peek User Testing, I saw there was confusion. Potential users didn't know how to intrept low fidelity that looked like an incomplete final product.
So, I quickly transformed low to hi-fidelity wireframes and began the user testing with a prototype that was in final state format. Great reviews...

"...one of the easiest checkout processes I've ever used!"
